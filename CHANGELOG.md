# Changelog

All notable changes to GS Center will be documented in this file.

## [2.2.9] - 2026-04-16
### Improved
- Full Cache Cleanup modal now automatically closes after completion instead of showing a Restart button.

## [2.2.8] - 2026-04-16
### Added
- Real-time download progress tracking with speed, size, and percentage.
- Cancel downloads mid-progress.

### Improved
- Redesigned Software Updates page with a cleaner single-row layout.
- Redesigned What's New as a centered modal with version timeline sidebar.

### Fixed
- Cancel status now shows the correct message.
- Floating progress bar now displays actual status instead of always showing "Update failed".

### Removed
- Removed redundant toast notifications from Software Updates.

## [2.2.7] - 2026-04-16
### Improved
- Rebuilt Software Updates with full real-time download progress (MB downloaded/total, speed, percentage).

### Fixed
- Fixed duplicate toast notifications during updates.
